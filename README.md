## **Python Greeting Program**

This is a simple Python program that asks the user for their name and prints a personalized greeting. It demonstrates basic Python concepts such as functions, user input, and console output.



## **Features**
- Reads user input  
- Generates a personalized greeting  
- Clean and beginner-friendly code  
- Simple CLI (Command Line Interface)  



## **How to Run**

1. Ensure you have **Python 3** installed.
2. Download or clone this repository.
3. Open your terminal or command prompt in the project folder.
4. Run the script:

```bash
python3 main.py
```



## **Example Output**

```
Enter your name: Vishal
Hello, Vishal! Welcome to the program.
```
